# orangepi_pc2_h5_gpio_python
OrangePi-H5-GPIO-Python驱动，通过移植 pyA20 0.2.1得来.


- 基于pyA20的开源修改

    - 参考 https://www.olimex.com/wiki/A20-OLinuXino-MICRO pyA20 https://pypi.python.org/pypi/pyA20
    
    
- 安装方法：

    - python setup.py install 
    
    
- 安装前提：
  
  -  只是用于PC2，其他不可以，或者说未添加CPU框架识别.
  
  -  需要安装 gcc g++ make 这类工具.
  
    - sudo apt install -y gcc g++ make
      
      
      
      
 目前在测试阶段，如果编译后还是无法运行的，请告之...
